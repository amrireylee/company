##企业信息管理系统

1. 简述
    * 该项目为企业信息管理系统。
    * 参考已有系统进行扩充以及重构，原来是SSH框架，扩展为spring+springMVC+Hibernate+mybatis的框架。
2. 功能
    * 已实现的：权限管理，员工信息管理，部门与职位管理，员工培训管理，考勤管理，薪酬管理；
    * 未实现功能：后勤管理，事务进度管理；各功能的优化，一些报表的导出，薪资、员工信息和考勤的联合增删改。
